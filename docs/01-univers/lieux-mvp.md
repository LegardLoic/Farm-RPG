# Lieux MVP

> **Statut : document canonique actif**
> Ce document définit les lieux canoniques du MVP.
> Il sert de référence active pour le front, la narration, les interactions, les PNJ, les scènes, les transitions et la mise en espace du jeu.
> En cas de divergence avec les anciens documents de cadrage ou les archives, ce document prévaut pour les lieux du MVP.

---

## 1. Rôle du document

Ce document fixe les lieux réellement nécessaires au MVP.

Il permet de :
- définir les espaces jouables ou visibles du prototype ;
- préciser leur fonction narrative et gameplay ;
- guider la transformation du front en vraie expérience visuelle ;
- relier les systèmes existants à des espaces concrets ;
- éviter que le monde reste un assemblage de panneaux HUD sans ancrage spatial.

Le MVP n’a pas besoin d’une carte immense.
Il a besoin d’un **petit ensemble de lieux clairs, mémorisables, utiles et fortement liés à la boucle ferme ↔ village ↔ tour**.

---

## 2. Principes de conception des lieux MVP

Les lieux du MVP doivent respecter les règles suivantes :

### 2.1 Lisibilité
Le joueur doit comprendre rapidement :
- où il est ;
- à quoi sert le lieu ;
- qui s’y trouve ;
- ce qu’il peut y faire.

### 2.2 Fonction mixte
Chaque lieu important doit avoir :
- une fonction gameplay ;
- une fonction d’ambiance ou de narration.

### 2.3 Échelle raisonnable
Le MVP doit éviter les lieux trop grands ou trop dispersés.
Le monde doit sembler vivant, mais rester maîtrisable.

### 2.4 Cohérence émotionnelle
Les lieux doivent soutenir la courbe du jeu :
- ferme = refuge et reconstruction ;
- village = quotidien blessé mais vivant ;
- tour = pression, hostilité, verticalité, malédiction.

---

## 3. Vue d’ensemble des lieux canoniques du MVP

Le MVP repose sur **3 pôles majeurs**, chacun décomposé en sous-zones simples.

| Lieu principal | Fonction dominante | Sous-zones MVP principales | Priorité |
|---|---|---|---|
| La Ferme | Base du joueur, production, repos | maison/abri, parcelles, zone de craft simple | Critique |
| Le Village | Hub narratif, social et économique | place ou axe principal, mairie, forge, marché, coin calme | Critique |
| La Tour | Progression combat, paliers, bosses | entrée, étages 1-10 en tranches d’ambiance | Critique |

---

## 4. La Ferme

## 4.1 Fonction générale

La Ferme est la base du joueur.

C’est le lieu où il :
- revient après ses boucles de progression ;
- cultive ;
- récolte ;
- dort ;
- transforme certaines ressources ;
- se réorganise avant de repartir.

La Ferme est l’espace le plus intime du MVP.

---

## 4.2 Rôle narratif

Narrativement, la Ferme représente :
- le premier lieu que le joueur peut vraiment remettre en état ;
- la preuve qu’un bout du monde peut redevenir productif ;
- un refuge imparfait mais essentiel ;
- l’espace où l’on sent le plus directement le passage de la stérilité au retour du vivant.

Elle ne doit pas être luxueuse.
Elle doit être modeste, un peu rude, mais porteuse d’espoir.

---

## 4.3 Rôle gameplay

La Ferme sert à :
- planter des graines ;
- arroser ;
- attendre la croissance ;
- récolter ;
- dormir ;
- accéder au craft simple ;
- préparer indirectement les combats.

C’est l’un des piliers mécaniques du projet.
Le front doit donc la rendre :
- claire ;
- manipulable ;
- agréable à lire ;
- visuellement distincte du village et de la tour.

---

## 4.4 Sous-zones canoniques de la Ferme

### A. L’abri / maison du joueur
Fonction :
- point de repos ;
- ancrage personnel ;
- lieu du sommeil ;
- possible point de sauvegarde visuel.

Ressenti recherché :
- simplicité ;
- protection ;
- débrouille ;
- début de foyer.

### B. Les parcelles cultivables
Fonction :
- cœur du gameplay agricole.

Ressenti recherché :
- terre reprise de haute lutte ;
- espace modeste mais concret ;
- progression visible selon ce qui pousse.

### C. Zone de travail / craft simple
Fonction :
- lieu où certaines récoltes deviennent consommables ou objets utiles.

Ressenti recherché :
- bricolage organisé ;
- préparation ;
- transformation utile plutôt qu’artisanat sophistiqué.

### D. Bordure de la ferme / environnement proche
Fonction :
- donner de l’air visuel ;
- situer la ferme dans le monde ;
- éviter l’effet “menu isolé”.

Ressenti recherché :
- campagne fragilisée mais encore habitable ;
- nature encore possible ;
- monde un peu vide, un peu silencieux.

---

## 4.5 Ambiance visuelle de la Ferme

La Ferme doit évoquer :
- la terre ;
- le bois ;
- le calme ;
- la reprise ;
- la modestie.

Palette recommandée :
- bruns naturels ;
- verts doux ;
- beige ;
- quelques tons ternis par la malédiction au départ.

Évolution attendue :
- plus la progression avance, plus la ferme doit paraître vivante et assumée.

---

## 5. Le Village

## 5.1 Fonction générale

Le Village est le hub social, économique et narratif du MVP.

C’est le lieu où le joueur :
- parle aux PNJ ;
- suit l’évolution du monde ;
- achète et vend ;
- observe les conséquences de sa progression ;
- revient entre deux poussées dans la tour.

Le Village ne doit pas être immense.
Il doit être **petit mais dense**.

---

## 5.2 Rôle narratif

Narrativement, le Village représente :
- le lieu à sauver ;
- le quotidien blessé par la malédiction ;
- la communauté qui survit encore ;
- la preuve visible que la progression du joueur sert à quelque chose.

Le Village ne doit pas sembler totalement mort.
Il doit sembler :
- ralenti ;
- fatigué ;
- suspendu ;
mais encore debout.

---

## 5.3 Rôle gameplay

Le Village sert à :
- rencontrer les PNJ principaux ;
- accéder au marché ;
- accéder à la forge ;
- suivre la progression relationnelle ;
- recevoir ou faire avancer certaines quêtes ;
- ressentir le retour monde après les paliers.

Le front doit donc le rendre :
- central ;
- lisible ;
- incarné ;
- suffisamment varié pour éviter l’effet “un seul écran de menu”.

---

## 5.4 Sous-zones canoniques du Village

### A. Entrée ou axe principal du village
Fonction :
- première lecture du lieu ;
- circulation simple entre les autres sous-zones.

Ressenti recherché :
- village modeste ;
- calme étrange ;
- impression de lieu qui vit au ralenti.

### B. La mairie / maison du Maire
Fonction :
- lieu lié à l’introduction ;
- ancrage du Maire ;
- point de suivi narratif principal.

Ressenti recherché :
- autorité locale modeste ;
- lieu sérieux mais usé ;
- centre administratif encore debout.

### C. La forge
Fonction :
- ancrage du Forgeron ;
- lieu de progression matérielle ;
- symbole fort de la reconstruction.

Ressenti recherché :
- atelier d’abord ralenti, presque éteint ;
- puis progressivement plus vivant, plus chaud, plus utile.

### D. Le marché / étal de la Marchande
Fonction :
- achat de graines ;
- vente des récoltes ;
- cœur du quotidien économique.

Ressenti recherché :
- petit commerce de survie au départ ;
- activité plus stable ensuite ;
- espace simple mais très important pour la boucle ferme ↔ village.

### E. Le coin calme / zone de l’Habitante secondaire
Fonction :
- interaction plus humaine ;
- respiration ;
- petits dialogues ou quêtes plus intimes.

Ressenti recherché :
- lieu discret ;
- un peu isolé ;
- propice à des scènes plus calmes.

### F. L’espace de l’Herboriste
Fonction :
- soutien du craft vivant ;
- lecture plus organique du village.

Ressenti recherché :
- plantes ;
- travail patient ;
- douceur fragile au milieu d’un lieu encore marqué par la crise.

---

## 5.5 Ambiance visuelle du Village

Le Village doit évoquer :
- la pierre ;
- le bois ;
- des lieux habités mais fatigués ;
- une vie locale qui continue par inertie.

Palette recommandée :
- gris doux ;
- bruns ;
- verts ternis ;
- tons neutres légèrement désaturés.

Évolution attendue :
- à mesure que le joueur progresse, le Village doit paraître :
  - moins figé ;
  - plus habité ;
  - plus utile ;
  - plus confiant.

---

## 6. La Tour

## 6.1 Fonction générale

La Tour est le pôle combat / progression du MVP.

C’est le lieu où le joueur :
- affronte les ennemis ;
- franchit les paliers ;
- rencontre les mini-boss et le boss majeur ;
- brise les premiers nœuds de la malédiction.

La Tour est la présence hostile dominante du monde.

---

## 6.2 Rôle narratif

Narrativement, la Tour représente :
- la source visible du mal ;
- le lieu qui domine le paysage ;
- l’axe de montée vers quelque chose de plus central ;
- une architecture de corruption et de défense.

Elle ne doit pas être un simple “donjon générique”.
Elle doit sembler :
- pesante ;
- verticale ;
- anormale ;
- de plus en plus oppressante à mesure qu’on monte.

---

## 6.3 Rôle gameplay

La Tour sert à :
- déclencher les combats ;
- organiser les paliers 3 / 5 / 8 / 10 ;
- fournir du loot, des matériaux et des objets clés ;
- rythmer la progression principale du MVP.

Le front n’a pas besoin de 10 décors complètement différents.
Mais il doit faire sentir :
- une montée de danger ;
- une montée d’étrangeté ;
- une montée d’oppression.

---

## 6.4 Sous-zones canoniques de la Tour

### A. Entrée de la Tour
Fonction :
- seuil symbolique entre le monde encore récupérable et le monde dominé par la malédiction ;
- point de départ visuel de l’ascension.

Ressenti recherché :
- malaise ;
- poids ;
- appel du danger.

### B. Étages 1 à 2 — bas de tour
Fonction :
- découverte ;
- premiers ennemis ;
- installation de l’étrangeté.

Ressenti recherché :
- danger lisible ;
- altération encore contenue ;
- première friction avec la corruption.

### C. Étage 3 — premier seuil
Fonction :
- premier cap fort ;
- premier gardien marquant.

Ressenti recherché :
- première vraie résistance ;
- début de la sensation de “nœud”.

### D. Étages 4 à 5 — zone de braise
Fonction :
- montée de pression ;
- ambiance plus agressive ;
- approche du Gardien des cendres.

Ressenti recherché :
- chaleur ;
- lourdeur ;
- usure ;
- hostilité plus active.

### E. Étages 6 à 7 — défense densifiée
Fonction :
- préparer le palier 8 ;
- faire sentir une menace plus structurée.

Ressenti recherché :
- tension plus froide ;
- menace plus ordonnée ;
- moins d’instinct, plus de garde.

### F. Étage 8 — seuil avancé
Fonction :
- dernier grand verrou avant le sommet MVP.

Ressenti recherché :
- hiérarchie ;
- défense active ;
- présence d’une volonté hostile plus claire.

### G. Étages 9 à 10 — approche du cœur
Fonction :
- préparation du boss final ;
- confrontation plus directe avec la malédiction.

Ressenti recherché :
- oppression ;
- gravité ;
- sentiment que l’on entre dans une zone plus “centrale” que simplement dangereuse.

### H. Étage 10 — boss majeur
Fonction :
- confrontation avec l’Avatar du Cœur de la Malédiction ;
- conclusion du MVP.

Ressenti recherché :
- rupture ;
- présence du noyau maudit ;
- victoire lourde de conséquences.

---

## 6.5 Ambiance visuelle de la Tour

La Tour doit évoluer visuellement selon une logique simple :

### Début
- pierre ancienne ;
- végétation déformée ;
- menace encore “vivante”.

### Milieu
- cendres ;
- chaleur ;
- structures plus dures ;
- sentiment de défense.

### Fin
- noirceur ;
- veines de malédiction ;
- architecture plus oppressante ;
- présence plus directe du cœur maudit.

Palette recommandée :
- gris sombres ;
- verts corrompus ;
- rouges de braise ;
- noirs profonds ;
- lueurs inquiétantes plus tardives.

---

## 7. Relations spatiales entre les lieux

Le MVP doit faire sentir une relation claire entre les trois grands pôles :

### Ferme ↔ Village
Proximité pratique.
Le joueur doit sentir qu’il vit “à côté” du village, mais pas au cœur de lui.

### Village ↔ Tour
Relation de domination.
La Tour pèse sur le Village, visuellement et symboliquement.

### Ferme ↔ Tour
Relation de contraste.
La Ferme produit, soigne, calme.
La Tour consomme, bloque, oppresse.

Cette lecture spatiale est très importante pour le front.

---

## 8. Fonction des lieux dans l’expérience joueur

Les lieux du MVP doivent structurer l’expérience ainsi :

### La Ferme
“Je reprends la main.”

### Le Village
“Je vois pour qui et pour quoi je me bats.”

### La Tour
“Je vais affronter ce qui bloque tout le reste.”

Si ces trois sensations existent clairement, alors le front commencera réellement à faire exister le jeu.

---

## 9. Contraintes de cohérence

Les lieux du MVP doivent toujours respecter les règles suivantes :

- chaque lieu doit avoir une fonction claire ;
- chaque sous-zone doit justifier son existence ;
- pas de lieux purement décoratifs sans utilité de mise en scène ou de gameplay ;
- le monde doit sembler petit, mais cohérent ;
- la progression doit modifier la perception des lieux, même subtilement.

Le joueur doit sentir que ces lieux forment un tout vivant et non trois menus séparés.

---

## 10. Contraintes front et mise en scène

Le front doit pouvoir s’appuyer sur ce document pour :
- placer les PNJ dans des lieux crédibles ;
- transformer certains panneaux HUD en interactions situées ;
- matérialiser les transitions entre les pôles ;
- créer des scènes simples mais ancrées.

Priorités de mise en scène recommandées :
- rencontre avec le Maire au village ;
- vue claire des parcelles à la ferme ;
- vrai point marché pour la Marchande ;
- vraie forge pour le Forgeron ;
- entrée identifiable de la Tour.

Le MVP n’a pas besoin de tout montrer avec un luxe extrême.
Il a besoin de montrer **les bons lieux au bon moment**.

## 13. Résumé exécutif

Le MVP repose sur trois grands lieux :
- **la Ferme**
- **le Village**
- **la Tour**

Chacun remplit une fonction claire et complémentaire :
- la Ferme = reconstruction et préparation ;
- le Village = vie locale, échanges, conséquences visibles ;
- la Tour = menace, progression, confrontation.

L’objectif de ces lieux n’est pas d’être immenses.
L’objectif est qu’ils soient **assez forts, assez lisibles et assez incarnés** pour que le joueur sente qu’il agit dans un monde réel, pas dans une simple juxtaposition de systèmes.
