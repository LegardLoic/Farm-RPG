# Guide visuel MVP

> **Statut : document canonique actif**
> Ce document définit la direction visuelle canonique du MVP.
> Il sert de référence active pour le front, les interfaces, les lieux, les PNJ, les monstres, les objets, l’ambiance générale et les futurs besoins de production graphique.
> En cas de divergence avec les anciens documents de cadrage ou les archives, ce document prévaut pour la direction visuelle du MVP.

---

## 1. Rôle du document

Ce document fixe la direction visuelle du MVP.

Il permet de :
- définir l’identité visuelle générale du projet ;
- donner une cohérence commune aux lieux, personnages, monstres, objets et UI ;
- guider les futures scènes front ;
- éviter une accumulation de styles contradictoires ;
- servir de base commune aux décisions de sprites, portraits, interfaces et ambiance.

Le MVP n’a pas besoin d’une bible artistique exhaustive de production AAA.
Il a besoin d’une **direction visuelle claire, forte, lisible et cohérente avec la boucle du jeu**.

---

## 2. Intention visuelle générale

Le jeu doit immédiatement évoquer un mélange entre :

- **RPG 2D top-down en pixel art**
- **jeu de ferme chaleureux mais modeste**
- **monde légèrement assombri par une malédiction**
- **tour inquiétante et progressivement oppressante**

La direction visuelle du MVP doit donc toujours chercher un équilibre entre :

### 2.1 Le familier
- ferme
- récoltes
- outils
- village
- gestes simples du quotidien

### 2.2 L’abîmé
- fatigue des lieux
- couleurs ternies
- bâtiments ralentis ou sous-exploités
- habitants usés

### 2.3 Le menaçant
- silhouettes hostiles
- corruption du vivant
- cendres
- cœur maudit
- verticalité oppressante de la tour

---

## 3. Promesse visuelle du MVP

Visuellement, le joueur doit ressentir :

- qu’il entre dans un petit monde encore vivant ;
- que ce monde est blessé mais récupérable ;
- que la ferme et le village ont une chaleur fragile ;
- que la tour tranche clairement avec le reste ;
- que plus il progresse, plus l’univers devient visuellement lourd, sombre et inquiétant.

Le visuel doit soutenir la promesse suivante :
**ce monde peut revivre, mais quelque chose au-dessus de lui l’empêche encore de respirer.**

---

## 4. Style graphique global

## 4.1 Type de rendu
Le MVP doit viser un rendu :

- **pixel art 2D top-down**
- lisible
- stylisé
- légèrement détaillé, sans excès
- plus évocateur que réaliste

## 4.2 Niveau de détail
Le niveau de détail recommandé est :

- suffisant pour différencier clairement les lieux et les PNJ ;
- assez simple pour garder une bonne lisibilité de gameplay ;
- assez expressif pour donner de la personnalité aux personnages et monstres ;
- pas trop chargé pour éviter la bouillie visuelle.

## 4.3 Philosophie visuelle
Le style doit privilégier :
- la lecture rapide ;
- les silhouettes ;
- les couleurs de fonction ;
- les contrastes d’ambiance.

Il faut éviter :
- le photoréalisme ;
- l’hyper-saturation constante ;
- les effets trop modernes ou trop brillants ;
- les interfaces “fantasy générique mobile”.

---

## 5. Ton visuel général

Le ton visuel du MVP doit mélanger :

### 5.1 Chaleur modeste
Pour :
- la ferme
- les récoltes
- certains espaces du village
- les interactions humaines

### 5.2 Fatigue du monde
Pour :
- les bâtiments
- les attitudes des PNJ
- l’atmosphère générale du village au début
- la sensation de malédiction diffuse

### 5.3 Tension croissante
Pour :
- la tour
- les paliers
- les bosses
- les effets liés à la corruption

La direction visuelle doit donc être capable de passer :
- du quotidien réparable
- à l’oppression croissante
sans rupture artificielle.

---

## 6. Direction visuelle des trois grands pôles

## 6.1 La Ferme

### Intention
La Ferme doit être visuellement :
- simple ;
- concrète ;
- un peu pauvre ;
- mais vivante et rassurante.

### Mots-clés visuels
- terre
- bois
- outils simples
- cultures visibles
- petit abri
- organisation modeste

### Ressenti
Le joueur doit voir la Ferme comme :
- un lieu personnel ;
- un refuge ;
- une base de reprise ;
- un espace qui répond à ses actions.

### Palette recommandée
- bruns naturels
- verts doux
- beige
- jaune paille discret
- quelques tons éteints au début

### Effet d’évolution
Au fil de la progression :
- la Ferme peut paraître plus entretenue ;
- plus stable ;
- plus productive ;
- plus chaleureuse.

---

## 6.2 Le Village

### Intention
Le Village doit être visuellement :
- petit ;
- crédible ;
- habité ;
- ralenti ;
- pas encore effondré.

### Mots-clés visuels
- pierre
- bois
- ruelles simples
- étal modeste
- forge fatiguée
- maisons closes ou semi-actives
- centre modeste, pas capitale

### Ressenti
Le joueur doit voir le Village comme :
- un lieu à sauver ;
- une communauté suspendue ;
- un endroit qui peut reprendre vie.

### Palette recommandée
- gris doux
- bruns
- verts ternis
- rouges discrets très désaturés
- touches chaudes localisées

### Effet d’évolution
À mesure que le joueur progresse :
- plus d’activité visuelle ;
- plus de chaleur ;
- plus de stabilité dans les lieux ;
- moins de sensation d’abandon silencieux.

---

## 6.3 La Tour

### Intention
La Tour doit être visuellement :
- plus sombre ;
- plus verticale ;
- plus hostile ;
- plus anormale ;
- plus oppressante à chaque palier.

### Mots-clés visuels
- pierre froide
- fissures
- ombres profondes
- végétation corrompue au bas
- braises et cendres au milieu
- structures plus maudites et plus organiques en haut

### Ressenti
Le joueur doit voir la Tour comme :
- le pôle de danger absolu du MVP ;
- quelque chose qui domine le monde ;
- un lieu qui semble vouloir retenir ou étouffer.

### Palette recommandée
- gris sombres
- verts corrompus
- rouges de braise
- noirs profonds
- violets ou rouges malsains sur la fin

### Effet d’évolution
Les étages doivent progressivement passer :
- du simplement altéré,
- au franchement hostile,
- puis au maudit presque central.

---

## 7. Direction visuelle des PNJ

## 7.1 Principes généraux
Les PNJ du MVP doivent être :

- lisibles en silhouette ;
- reconnaissables rapidement ;
- visuellement cohérents avec leur rôle ;
- expressifs sans surjeu caricatural.

Chaque PNJ principal doit avoir :
- une silhouette identifiable ;
- une palette dominante légère ;
- une posture cohérente avec sa fonction ;
- un ou deux états visuels si possible.

## 7.2 Ligne de lecture recommandée

### Le Maire
- tenue sobre
- autorité locale modeste
- fatigue digne
- tons neutres / terreux

### Le Forgeron
- carrure forte
- tablier / métal / chaleur
- posture lourde
- tons plus sombres et chauds

### La Marchande
- pratique
- mobile
- plus ouverte
- marché / sacs / paniers
- tons plus doux, commerce de proximité

### L’Habitante secondaire
- simplicité
- douceur
- retenue
- présence calme
- palette plus discrète

### L’Herboriste
- calme
- plantes / sachets / séchage / fioles simples
- tons végétaux doux
- lecture organique et apaisée

---

## 8. Direction visuelle des monstres

## 8.1 Principes généraux
Les monstres du MVP doivent être :
- reconnaissables immédiatement ;
- lisibles en combat ;
- cohérents avec leur palier ;
- plus marquants par silhouette et ambiance que par surcharge de détails.

## 8.2 Progression visuelle recommandée

### Bas de tour
- créatures encore “vivantes”
- gobelins, mannequins, bêtes altérées
- déformation compréhensible
- corruption encore périphérique

### Palier 3
- vivant densifié
- végétal hostile
- épines, excroissances, masse animale corrompue

### Palier 5
- braise, charbon, chaleur
- gardien plus massif
- structure plus stable, moins bestiale

### Palier 8
- hiérarchie
- forme plus humanoïde / commandement corrompu
- discipline hostile

### Palier 10
- présence maudite
- forme plus centrale, plus dérangeante
- moins “monstre”, plus “manifestation”

---

## 9. Direction visuelle des objets

## 9.1 Principes généraux
Les objets du MVP doivent être :
- lisibles en petit format ;
- distincts par catégorie ;
- cohérents avec leur fonction ;
- simples à reconnaître dans l’inventaire et les shops.

## 9.2 Lecture par famille

### Consommables
- couleurs claires et fonctionnelles
- lecture immédiate de l’effet
- herbe = végétal / soin
- tonique = fiole / préparation / mana

### Graines et récoltes
- visuels naturels
- très lisibles
- correspondance claire entre graine et récolte

### Matériaux
- plus bruts
- plus minéraux, organiques ou corrompus
- bien différenciés des objets consommables

### Objets clés
- plus marquants
- plus “importants”
- lecture narrative plus forte
- ne doivent pas ressembler à de simples matériaux

### Équipements
- lisibilité de slot
- style crédible avec le Forgeron et le village
- progression visuelle claire entre base et avancé

---

## 10. Direction visuelle de l’interface

## 10.1 Principes UI
L’interface du MVP doit être :
- claire ;
- lisible ;
- discrète sans être fade ;
- cohérente avec le monde ;
- plus “RPG artisanal” que “dashboard technique”.

## 10.2 Ce qu’il faut viser
- panneaux lisibles
- hiérarchie visuelle claire
- informations utiles bien groupées
- éléments de style rappelant le monde sans le surcharger

## 10.3 Ce qu’il faut éviter
- HUD trop technique ou trop “debug”
- interfaces trop froides
- excès de cadres décoratifs fantasy
- bruit visuel inutile

## 10.4 Mots-clés UI
- sobriété
- lisibilité
- matière légère
- fantasy modeste
- ergonomie d’abord

---

## 11. Effets visuels et feedbacks

Les feedbacks visuels du MVP doivent être :
- lisibles ;
- rapides à comprendre ;
- plus utiles que spectaculaires.

### À valoriser
- dégâts
- soin
- statut
- déblocage de shop
- montée de palier
- récompense de boss
- progression du monde

### À éviter
- excès de particules
- surcharge lumineuse
- animations longues ou confuses

Le front doit faire sentir les événements importants sans perdre la lisibilité.

---

## 12. Contrastes visuels majeurs du MVP

Le guide visuel doit préserver ces contrastes :

### Ferme vs Tour
- vivant / oppressant
- terre / pierre maudite
- reconstruction / consommation

### Marché vs Forge
- quotidien / matériel
- circulation / fabrication
- chaleur humaine / chaleur de travail

### Village vs Tour
- petit monde humain / force verticale hostile
- ralentissement / écrasement
- quotidien / seuil

Ces contrastes sont essentiels pour que le jeu ait une vraie identité visuelle.

---

## 13. Référentiel d’ambiance visuelle

Le jeu doit évoquer une combinaison entre :

- **RPG top-down lisible**
- **jeu de ferme modeste**
- **village touché mais pas détruit**
- **tour inquiétante et maudite**
- **fantasy simple mais pas générique**
- **ambiance douce dans le quotidien, plus sombre dans la progression**

Le mot-clé global à garder est :

**“chaleur fragile sous ombre grandissante”**

---

## 14. Contraintes de cohérence

La direction visuelle du MVP doit toujours respecter les règles suivantes :

- le monde doit rester lisible ;
- la ferme, le village et la tour doivent être immédiatement distinguables ;
- les PNJ doivent être reconnaissables sans surcharge ;
- les monstres doivent être hiérarchisés visuellement par palier ;
- les objets doivent être compréhensibles en petit format ;
- l’UI doit servir le jeu, pas l’écraser.

La cohérence visuelle doit primer sur la variété gratuite.

---

## 15. Utilité concrète pour la production

Ce document doit permettre ensuite de produire plus facilement :

- les maquettes d’écrans et scènes front ;
- les besoins en portraits PNJ ;
- les besoins en sprites de lieux et props ;
- les besoins en icônes d’objets ;
- les variations d’ambiance selon les zones ;
- les premiers choix de palette et d’interface.

Il doit servir de base, pas de prison.
Mais toutes les futures décisions visuelles doivent rester compatibles avec lui.

## 18. Résumé exécutif

La direction visuelle du MVP doit reposer sur un équilibre clair :

- **la Ferme** = refuge modeste et vivant
- **le Village** = quotidien blessé mais encore humain
- **la Tour** = oppression croissante et malédiction incarnée

Le style doit être en pixel art 2D top-down, lisible, stylisé, cohérent et émotionnellement clair.
Il doit faire sentir une progression visuelle simple mais forte :
**du réparable vers le menaçant, du quotidien vers le maudit, de la reprise fragile vers la confrontation avec un cœur sombre.**
