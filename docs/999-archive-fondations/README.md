# Archives de fondation

Ce dossier conserve les documents ayant servi au cadrage et à la construction initiale du projet.

## Rôle
- garder une trace des décisions fondatrices ;
- documenter l’historique du projet ;
- permettre de retrouver les hypothèses et arbitrages d’origine.

## Important
Ces documents ne constituent pas la source de vérité actuelle du MVP.

Pour la production en cours, se référer en priorité :
- aux documents canoniques actifs dans `docs/00-projet/` ;
- aux documents de contenu dans les dossiers thématiques ;
- au journal actif dans `docs/journal/journal-phase-2-contenu-front.md`.
